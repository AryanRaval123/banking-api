# Banking REST API

A complete Spring Boot banking API with account management, transactions, and fund transfers.

## Features
- ✅ Create and manage bank accounts
- ✅ Deposit and withdraw funds  
- ✅ Transfer money between accounts
- ✅ RESTful API design
- ✅ MySQL database
- ✅ Proper error handling

## Tech Stack
- Java 17
- Spring Boot 3.x
- Spring Data JPA
- MySQL Database
- Maven
